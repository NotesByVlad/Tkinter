Rock Paper Scissors Game

A simple Rock Paper Scissors game built using Python and Tkinter.

🎮 Features

Interactive GUI built with Tkinter

Player vs Computer gameplay

Dynamic score tracking

Reset button to restart the game

Custom images for choices and buttons

Styled interface with colors and fonts

🛠️ Installation

1. Clone the repository

Clone the repository 

cd Rock-Paper-Scissors

2. Install dependencies

Ensure you have Python 3.x installed.

pip install tk

3. Run the game

python main.py

📂 Project Structure

Rock-Paper-Scissors/
│── images/                # Folder for images (rock, paper, scissors, buttons)
│── main.py                # Main script for running the game
│── README.md              # Project documentation
